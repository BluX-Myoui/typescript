# TypeScript — Práctica 4 (Módulos 1 y 2)

Repositorio independiente para los laboratorios de TypeScript. El módulo 3 (React) vive en el repositorio `react`.

## Estructura

```
typescript/
├── modulo-1/    # Laboratorio 1: inicialización y lógica pura
│   ├── src/
│   ├── dist/
│   └── docs/
└── modulo-2/    # Laboratorio 2: modelado de datos y genéricos
    ├── src/
    └── docs/
```

## Arranque rápido

**Módulo 1**
```bash
cd modulo-1
npm install
npm run dev
npm run build
```

**Módulo 2**
```bash
cd modulo-2
npm install
npm run dev
npx tsc --noEmit
```
